This tool does SSG.

* [MD and YAML Cheatsheet](https://learntheweb.courses/topics/markdown-yaml-cheat-sheet/#yaml)
* [Jekyll Website](https://jekyllrb.com/)
* [Jekyll Github Repo](https://github.com/jekyll/jekyll)

* This is setup to do work and not have the mechanics visible on the github repo. Build in this repo. Sync other.
* Edit _config.yml, index.md and execute jekyll to convert to static webpages.
* Run '$ bundle exec jekyll build' or '$ bundle exec jekyll b' to build to _site/.
* Cleanup files from _site/ that shouldn't be posted.
* Copy _site/ to jtvisona.github.io folder.
